## Comandos

- **`echo I am user $myname`**:
    
    - Este comando `echo` imprime la cadena de texto `"I am user $myname"`. La variable `$myname` se expandirá a su valor actual. Por ejemplo, si `$myname` es `Alice`, el comando imprimirá `I am user Alice`.
- **`md5sum`**:
    
    - `md5sum` es una utilidad que calcula el hash MD5 de la entrada proporcionada. En este caso, tomará la cadena de texto producida por el primer `echo` y calculará su hash MD5.
- **`cut -d ' ' -f 1`**:
    
    - `cut` se usa para extraer secciones de texto. En este caso, se utiliza con `-d ' '` para especificar que el delimitador es un espacio y `-f 1` para seleccionar el primer campo. `md5sum` produce una salida con el hash MD5 seguido de un espacio y el nombre del archivo (o, en este caso, el texto), por lo que `cut` selecciona solo el hash MD5.
- **`echo $(...)`**:
    
    - La parte exterior `echo $(...)` toma el resultado del comando dentro de `$(...)`, que es el hash MD5 calculado, y lo imprime.
